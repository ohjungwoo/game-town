=== HTTP status codes

RESTful 방식의 응답코드 정리표 입니다.
GameTown 에서는 아래와 같은 응답코드 위주로 사용합니다.

|===
| Status code | Usage

| `200 OK`
| 성공


| `400 Bad Request`
| GameTown 에서, 사용자 입력 validation 관련 모든 처리는
400 에러로 반환합니다.

| `500 Internal Error`
| 예상하지 못한 에러입니다. 발생 시, 개발팀에 문의주세요.
|===
